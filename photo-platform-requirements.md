# 摄影作品分享平台 — 需求规格说明书

---

## 一、使用技术

```
Python + MySQL + Flask + HTML + CSS + JS + Bootstrap + Layer + jQuery  + 阿里云 OSS
```

---

## 二、系统角色

| 角色 | 说明 |
|------|------|
| 用户端 | 浏览作品、搜索摄影师、点赞收藏评论、论坛发帖交流、 |
| 管理员端 | 审核作品/摄影师、管理用户、论坛/评论管理、数据可视化、系统配置 |

---

## 三、主要完成的任务

1. 在管理员端使用可视化工具实现对**作品、摄影师、作品分类**等相关数据以图表化形式展示

2. 管理员可以对**作品、摄影师、论坛、评论、公告、用户**等信息进行管理

3. 用户可以查看管理员发布的**精选作品、推荐摄影师**等信息

4. 用户可以在论坛进行发帖，其他用户可以评论

5. 用户可以根据关键词查询作品

6. 使用阿里云 OSS 对用户上传的头像、用户在论坛中发布的帖子中的图片、管理员在作品详情中发布的组图、公告中发布的图片进行存储

---

## 四、8 大功能模块

### 模块 1：登录管理
- 用户注册（用户名 + 密码 ，角色选择：普通用户 / 摄影师）
- 用户登录 / 退出
- 密码 MD5 加密存储
- 管理员独立登录入口

### 模块 2：控制台
- 系统概况卡片（总作品数、总用户数、活跃摄影师数）
- 最近 7  天新增用户与新增作品双轴折线图
- 热门作品 Top 10 排行榜（按浏览量 + 点赞 + 收藏加权）
- 最新注册摄影师列表（待审核）

### 模块 3：作品管理
- 作品列表（分页、搜索、筛选）
- 作品详情（标题、摄影师、拍摄城市、组图轮播 OSS、描述、热度、点赞数）
- 作品新增 / 编辑 / 删除（管理员可审核下架）
- 作品组图 → 阿里云 OSS 存储
- 作品热度算法（浏览量 、 点赞 、 评论加权算法）

### 模块 4：摄影师管理
- 摄影师列表（分页、搜索、筛选）
- 摄影师详情（头像 OSS、作品集、所在城市、简介）
- 摄影师认证审核（管理员审核摄影师注册申请）
- 摄影师作品集主页（自动聚合该摄影师所有作品）

### 模块 5：分类与标签管理
- 风格标签（日系小清新、复古胶片、黑白光影、森系、ins风、情绪片）
- 风格标签新增 / 编辑 / 删除

### 模块 6：用户管理
- 用户列表（分页、搜索、按角色筛选：普通用户 / 摄影师）
- 用户详情（注册时间）
- 摄影师认证审核
- 个人中心：修改头像（OSS）、修改密码、我的帖子

### 模块 7：互动管理
- 论坛板块（作品交流、技巧分享）
- 用户发帖（标题 + 内容 + 图片 → OSS + 所属板块）
- 帖子列表（分页、搜索、按热度/时间排序）
- 帖子详情 + 点赞
- 评论
- 管理员可删帖 / 删评论 / 置顶
- 作品评论区

### 模块 8：可视化与系统管理

#### 8.1 可视化大屏
- 作品风格标签分类占比饼图
- 月度新增用户与新增作品双轴趋势图
- （可选）论坛活跃度趋势（发帖量 / 评论量折线图）
- 风格标签词云

#### 8.2 系统管理
- 公告管理（发布 / 编辑 / 删除，富文本 + OSS 图片）
- 首页精选置顶
- 轮播图 
- 系统操作日志
