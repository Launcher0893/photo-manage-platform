摄影作品分享平台
使用技术:
	python + mysql + flask + html + css + js + bootstarp + layer + jquery
系统角色:
	用户端
	管理员端
主要完成的任务:
	1、在管理员端使用可视化工具实现对作品、摄影师、作品分类等相关数据以图表化形式展示
	2、管理员可以对作品、摄影师、论坛、评论、公告、用户等信息进行管理
	3、用户可以查看管理员发布的精选作品、推荐摄影师等信息
	4、用户可以在论坛进行发帖，其他用户可以评论
	5、用户可以根据关键词等查询作品。
	6、使用阿里云 OSS 对用户上传的头像、用户在论坛中发布的帖子中的图片、管理员在作品详情中发布的组图、公告中发布的图片进行存储。
